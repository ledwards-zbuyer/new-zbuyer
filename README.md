# new-zbuyer

Concept mockups for a re-envisioned **zBuyer.com** — a consumer-facing (home seller) public site with a clear path to the Agents & Investors section.

## What's here

Three design directions, each as a full homepage + matching Agents/Investors area, responsive (desktop + mobile):

| Style | Homepage | Color |
|-------|----------|-------|
| Classic | `mockups/option-1-classic.html` | Trust blue + coral CTA |
| Apple Minimal | `mockups/option-2-apple-minimal.html` | Terracotta / clay |
| Dark Premium | `mockups/option-5-dark.html` | Midnight cyan |

Agents/Investors pages (re-envisioned `welcome.zbuyer.com`, "Leads. Simplified."):
`mockups/agents-classic-blue.html`, `mockups/agents-apple-gold.html`, `mockups/agents-dark-cyan.html`

- `mockups/assets/` — brand logo variants (recolored per style) and shared styles
- `mockups/shots/` — rendered desktop + mobile screenshots
- `logo/` — original zBuyer brand logo source files

## Key behavior

- Homepage hero = seller **address entry** (links to `pulse.zbuyer.com` selling funnel)
- Secondary **Buy a Home** option (links to the buying funnel)
- Prominent **For Agents & Investors** link → the matching agents page

Open any `.html` directly in a browser to view.
